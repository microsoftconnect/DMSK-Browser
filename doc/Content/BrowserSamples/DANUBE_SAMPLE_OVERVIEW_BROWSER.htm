Use cases supported by Dragon Medical SpeechKit

  • Dictation

Select the Record button, select a field and start speaking. The speech recognition system starts listening; turning speech into text.

  • What can I say

Say what can I say to open the personalization & help window. All voice commands and auto-texts available to the user are listed. The personalization & help window can also be opened via a button on the user interface.

  • Open/close help

Say open help/close help to open/close the help. The online help describes how to use the speech recognition system. The help can also be opened and closed via a button on the user interface.

  • Manage vocabulary

This feature helps you add words that aren't in the system. Say edit vocabulary to open the Manage Vocabulary page. To add a word, enter the Written form and Spoken form of the word and select Add.

  • Auto-texts

Auto-texts are predefined blocks of text that save time as you don't have to say the whole text every time. Say insert and the name of the auto-text to insert an auto-text. The available auto-texts are listed in the personalization & help window.

  • Correction

You can edit what is written in the fields via voice command. For example, say scratch that to delete the last thing you said. The available voice commands are listed in the personalization & help window.

  • Navigation

You can move the speech focus from one field to another (without using your cursor) via voice command. For example, say next field to move the speech focus to the next field. The available voice commands are listed in the personalization & help window.
