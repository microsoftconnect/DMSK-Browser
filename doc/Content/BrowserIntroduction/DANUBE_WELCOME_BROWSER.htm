Welcome to the Dragon Medical SpeechKit Browser edition help

Dragon Medical SpeechKit enables you to add speech recognition functionality to your web browser apps. We've designed it to be easy for you to integrate and for end users to use.

The service provided by Dragon Medical SpeechKit is guaranteed to be secure and usable wherever the user is working.

For more information on the new features in this version, see the release notes provided with the SDK package.

When your web browser app is speech-enabled your users can use a variety of our features. For information on the speech recognition features available, what the user sees on screen and how to use the features, see: End user documentation. For information on general concepts, see: Glossary.

Organization of this documentation

This documentation contains the following sections:

  • Concepts - information on the product's features and capabilities.
  • Setting up Dragon Medical SpeechKit - describes the package that we deliver.
  • Release channels - information on connecting to the Mainline, Beta and Delayed versions of the product.
  • Speech-enabling your browser app - the steps involved to get started with Dragon Medical SpeechKit.
  • Use cases - functionality that you can implement in your browser app using Dragon Medical SpeechKit.
  • Microphone button handling - provides information on supported control devices.
  • Technical restrictions - provides information that you should be aware of when integrating our package.
  • Release history - an archive of previous versions' release notes.
