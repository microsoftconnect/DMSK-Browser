Release Notes for 360 | SpeechAnywhere Services 1.4 (Browser edition)

Important information

  • 360 | SpeechAnywhere Services 1.4 Server is compatible with client versions 1.2 and 1.3.
  • The delete field voice command has been removed.
  • In NMC "customer token" has changed to "organization token".

  • Hidden or disabled fields, i.e. fields with the disabled attribute or the CSS properties visibility:hidden or display:none, are now excluded from speech recognition by default.

Known issues

  • The SpokenText parameter in the CommandRecognized event is empty in command mode. [PSP00062085]

Infrastructure no longer supported

  • Version 1.0 client-based applications will stop working.
  • Mozilla Firefox 16.x and lower
  • Google Chrome 23.x and lower

Infrastructure changes planned for the next release

  • Support for Mozilla Firefox 17.x and lower will be removed.
  • Support for Google Chrome 24.x and lower will be removed.
  • Support for Apple Safari 5.x will be removed.
  • In 360 | SpeechAnywhere Services 1.5 all version 1.1 client-based applications will stop working.
