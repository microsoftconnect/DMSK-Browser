Release Notes for 360 | SpeechAnywhere Services 1.4 Service Release 3 (Browser edition)

Solved issues

In Microsoft Internet Explorer selecting text via mouse now works properly. [PSP00069229]

In Microsoft Internet Explorer content editable controls, a new line is no longer added at the beginning of the field when recognized text is inserted. This issue is only solved if compatibility mode is enabled in Internet Explorer. [PSP00064624]

CKSource CKEditor integration has been improved. [PSP00067131]

The javascript no longer changes the height of the HTML BODY element; web pages are displayed properly. [PSP00069140]

Samples now reference version 1.4 URLs; documentation has been updated with the corresponding paths. [PSP00069125]

The 'Technical restrictions' topic in the SDK help has been updated. [PSP00069132]

The speech bar is only visible if at least one control is speech-enabled or at least one application or custom command is available. [PSP00069480]

The SDK now only registers an event handler for the focus events of speech-enabled elements (and custom containers). [PSP00069535]

The SDK deployment has been changed to make sure that the 360 | SpeechAnywhere Services Silverlight control and the 360 | SpeechAnywhere Services JavaScript file, as well as other SAS Browser edition files are always loaded in the correct version. This ensures there are no problems with browser caches or caching proxies. The Nuance.SpeechAnywhere.css file should no longer be directly included. [PSP00069558]

Important information

  • 360 | SpeechAnywhere Services 1.4 Server is compatible with client versions 1.2 and 1.3.
  • The delete field voice command has been removed.
  • In NMC "customer token" has changed to "organization token".
  • Hidden or disabled fields, i.e. fields with the disabled attribute or the CSS properties visibility:hidden or display:none, are now excluded from speech recognition by default.

  • NUSA_recordButtonHeight and NUSA_recordButtonWidth are deprecated and will be removed in 360 | SpeechAnywhere Services 1.5.

Known issues

  • The SpokenText parameter in the CommandRecognized event is empty in command mode. [PSP00062085]

Infrastructure no longer supported

  • Version 1.0 client-based applications will stop working.
  • Mozilla Firefox 16.x and lower
  • Google Chrome 23.x and lower

Infrastructure changes planned for the next release

  • Support for Mozilla Firefox 17.x and lower will be removed.
  • Support for Google Chrome 24.x and lower will be removed.
  • Support for Apple Safari 5.x will be removed.
  • In 360 | SpeechAnywhere Services 1.5 all version 1.1 client-based applications will stop working.
