Release Notes for Dragon Medical SpeechKit 3.1 Service Release 7 (Browser edition)

Solved issues in service release 7

  • Recording no longer occasionally stops unnecessarily. [DNB-9106]

Important information

  • Dragon Medical SpeechKit 3.1 (Browser edition) requires Dragon Medical Server 3.0 or 3.1. Make sure to upgrade the server before you upgrade to this Browser edition.
  • Dragon Medical Server 3.1 is compatible with Browser editions 3.0 and 3.1.
  • Do not change or hide Browser edition GUI elements. Changing or hiding these elements risks the stability and compatibility of the product; Nuance cannot provide support for such integrations.
  • Web browser debugging consoles or debugging proxies (for example, Fiddler) cannot be used with speech recognition. The performance impact incurred by such infrastructure leads to unreliable speech recognition.

What's new in version 3.1

Speech recognition

  • Speech recognition now uses neural network (deep learning) technology. (Only some languages are supported.)

Dragon Medical Analytics

  • The Server Management page is now displayed as a carousel in kiosk mode.
  • It is now possible to start Dragon Medical Analytics directly from the Nuance Management Center; this only applies to Nuance-hosted systems.
  • Dragon Medical Server configuration changes and PHI data access are now audited and accessible via the Nuance Management Server Auditor.
  • It is now possible to search for session logs based on the document ID provided by the partner.

User Interface

  • The personalization & help window has been redesigned to improve usability and to prepare it for additional features.
  • Auto-texts with lists are now supported as part of Dragon Medical PowerPacks. The auto-text contains variable text linked to a list of content options.
  • The speech recognition on/off icons now differ in both color and shape to support people with color vision deficiency.

On-premises systems

  • Dragon Medical Server can now connect to the Microsoft SQL Server using SQL Server Authentication.
  • Dragon Medical Server now supports Microsoft Windows Server 2016. Be aware that Nuance Management Server does not support Windows Server 2016.
  • Italian is now available as a user interface language.
  • Italian is now available as a speech recognition language for non-SpeechMagic-based systems.

Citrix environments

  • Nuance Citrix Server & Virtual Desktop Audio Extension is no longer available; the server binaries required are included in the application folder. Nuance Citrix Client Audio Extension does not need to be installed on the server/virtual desktop.
  • The Citrix extension deployment is now supported in a mixed environment (a range of client end points; for example, Linux thin clients and Windows clients).

Known issues

  • In Google Chrome audio is being recorded but is not properly processed. This is a Chrome issue; for more information, see: https://bugs.chromium.org/p/chromium/issues/detail?id=649377. [DNB-7992]
  • This release does not support mobile devices; including browsers (Firefox and Chrome) that do not use Silverlight.
  • iOS mobile devices do not work using any browser due to Apple limitations on third-party browsers (https://developer.chrome.com/multidevice/ios/overview) and lack of the corresponding API support, in Safari, by Apple (http://caniuse.com/#feat=stream%26feat_search=getUserMedia).

Infrastructure

Infrastructure no longer supported

  • Dragon Medical SpeechKit (Browser edition) 3.1 applications do not work with any server version prior to 3.0.
  • Mozilla Firefox 44 and lower are no longer supported (see Mozilla Firefox ESR Overview: https://www.mozilla.org/en-US/firefox/organizations/faq/).
  • Google Chrome 47 and lower are no longer supported.
  • In Dragon Medical Server 3.1 all version 1.5 client-based applications stop working.

Infrastructure changes planned for the next release

  • In Dragon Medical Server 3.2 all version 1.6 and 2.0 client-based applications will stop working.
  • Google Chrome 52 and lower will no longer be supported.

Remarks

  • Mozilla Firefox: Firefox for OS X is only supported with an external microphone.

  • Microsoft Internet Explorer on Windows:

    Internet Explorer is only supported with the Compatibility View and Protected Mode disabled. Additionally, some <!doctype> definitions are not supported; use <!doctype html>.

    For Windows Server 2012, Internet Explorer 10 is supported because Internet Explorer 11 is not available.

  • Microsoft Silverlight is only required for Microsoft Internet Explorer on Microsoft Windows and Apple Safari on OS X.

*End-of-life: third party environments

Support for third party environments is only valid as long as they are supported by the corresponding vendor and might be subject to other restrictions. Please contact Nuance Technical Support for details. For more information, see the documentation delivered with the third party product and supporting Nuance documentation.

When support by the vendor has stopped, Nuance will continue support if an issue is specific to the Nuance solution, within the limitations of the vendor’s end-of-life and Nuance's policy. This means that issues that are a combination of the vendor’s component and the Nuance solution cannot be supported.
