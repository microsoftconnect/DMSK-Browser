Release Notes for Dragon Medical SpeechKit 3.2 Release 1 (Browser edition)

Important information

  • Dragon Medical SpeechKit 3.2 Release 1 (Browser edition) requires Dragon Medical Server 3.0, 3.1 or 3.2. Make sure to upgrade the server before you upgrade to this Browser edition.
  • Dragon Medical Server 3.2 is compatible with Browser editions 3.0, 3.1 and 3.2.
  • Do not change or hide Browser edition GUI elements. Changing or hiding these elements risks the stability and compatibility of the product; Nuance cannot provide support for such integrations.
  • Web browser debugging consoles or debugging proxies (for example, Fiddler) cannot be used with speech recognition. The performance impact incurred by such infrastructure leads to unreliable speech recognition.

New in version 3.2

  • The Browser edition (Chrome on Windows only) now provides microphone button support for Nuance-hosted systems.

  • Browser edition user notifications have been reworked and are now less intrusive.

User Interface (requires Dragon Medical Server 3.2)

  • Dragon-based systems: A video tutorial is now available for new users with US English set as a speech recognition language.
  • The personalization & help window has been redesigned to improve consistency and conciseness.
  • The user interface is now localized in all speech recognition languages. If both the user interface language and speech recognition language are set, the user interface language will be ignored to provide a more consistent user experience.

On-premises systems

  • Dragon Medical Server 3.2 now supports SQL Server 2016.

Known issues

  • This release does not support mobile devices; including browsers (Firefox and Chrome) that do not use Silverlight.
  • iOS mobile devices do not work using any browser due to Apple limitations on third-party browsers (https://developer.chrome.com/multidevice/ios/overview) and lack of the corresponding API support, in Safari, by Apple (http://caniuse.com/#feat=stream%26feat_search=getUserMedia).

Infrastructure

Infrastructure no longer supported

  • Dragon Medical SpeechKit (Browser edition) 3.2 applications do not work with any server version prior to 3.0.
  • Mozilla Firefox 44 and lower are no longer supported (see Mozilla Firefox ESR Overview: https://www.mozilla.org/en-US/firefox/organizations/faq/).
  • Google Chrome 52 and lower are no longer supported.
  • In Dragon Medical Server 3.2 all version 1.6 and 2.0 client-based applications stop working.

Infrastructure changes planned for the next release

  • In Dragon Medical Server 3.3 all version 2.1 client-based applications will stop working.
  • Google Chrome 55 and lower will no longer be supported.

*End-of-life: third party environments

Support for third party environments is only valid as long as they are supported by the corresponding vendor and might be subject to other restrictions. Please contact Nuance Technical Support for details. For more information, see the documentation delivered with the third party product and supporting Nuance documentation.

When support by the vendor has stopped, Nuance will continue support if an issue is specific to the Nuance solution, within the limitations of the vendor’s end-of-life and Nuance's policy. This means that issues that are a combination of the vendor’s component and the Nuance solution cannot be supported.
